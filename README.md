# BMI Calculator - OIBSIP Task 1

## Objective
To develop a BMI Calculator using Python and Tkinter that calculates Body Mass Index based on user height and weight and displays the corresponding health category.

## Tools Used
- Python
- Tkinter
- VS Code

## Features
- User-friendly GUI
- Dark mode interface
- Name and Age input
- BMI calculation
- Health category classification
- Supports height in cm and meters

## Steps Performed
1. Created the GUI using Tkinter.
2. Added input fields for name, age, weight, and height.
3. Implemented BMI calculation logic.
4. Added BMI category classification.
5. Designed a dark-themed interface.
6. Tested the application with different inputs.

## Outcome
A fully functional BMI Calculator that calculates BMI accurately and displays health status through an attractive graphical interface.

## How to Run
1. Install Python.
2. Download the project files.
3. Run:
   python bmicalculator.py
